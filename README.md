# git analysis

## Usage

Create a JSON file to input.

Then run the script:

```sh
node main.js input.json
```

And wait for the processing the happen...

## Global behaviours

Some files are globally ignored for speed:

- `.tests.js$`
- `tests/`
- `i18n/`
- `.phan/`
- `.github/`

In the future these might also be configurable...
